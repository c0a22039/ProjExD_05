# ゲーム のタイトル

* 恐竜ゲーム

## 実行環境の必要条件

* python 3.10
* pygame >= 2.1

## ゲームの概要

Googleの恐竜ゲームのpygame版です。ジャンプやしゃがみで障害物を避けるゲームです。

## ゲームの実装


### 共通基本機能

* 恐竜が走る
* ジャンプする
* しゃがむ

### 担当追加機能

* スコアの追加
* 障害物の追加(担当：奥脇)
　障害物のサボテンを追加する。
　LargeCactusクラス：大きいサボテンを追加
　SmallCactusクラス：小さいサボテンを追加
* 障害物の追加2
* 背景装飾
* サウンドの追加
