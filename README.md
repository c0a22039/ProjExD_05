# ゲーム のタイトル

* 恐竜ゲーム

## 実行環境の必要条件

* python 3.10
* pygame >= 2.1

## ゲームの概要

Googleの恐竜ゲームのpygame版です。ジャンプやしゃがみで障害物を避けるゲームです。

## ゲームの実装

### 共通基本機能

* 恐竜が走る
* ジャンプする
* しゃがむ
